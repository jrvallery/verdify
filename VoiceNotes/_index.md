## About This Index

This master index provides a searchable overview of all processed voice notes. Each entry includes:

- **Date**: When the note was processed
- **Title**: Extracted or generated title
- **Duration**: Length of the recording in MM:SS format
- **Participants**: Names of people involved in the conversation
- **Tags**: Categorization tags for easy filtering
- **Note**: Link to the full structured note

## Search Tips

Use your text editor's search function to find notes by:
- Keywords in titles
- Participant names
- Tag names
- Date ranges

For more advanced queries, use the machine-readable `_index.json` file with tools like `jq`.

## Deduplication

Notes are deduplicated based on audio content (PCM hash), not file hash. This means:
- Same audio in different formats (MP3 vs M4A) will create only one entry
- Identical recordings will be detected regardless of filename
- Re-encoding or format conversion won't create duplicates
